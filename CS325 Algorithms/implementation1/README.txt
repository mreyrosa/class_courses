
To run these programs, compile the program by typing:
  g++ "program_name" -o "executable_name"

  example: g++ bruteforce.cpp -o bruteforce


Then run the program by typing:
  "executable_name" "text_file"

  example: bruteforce mytext
The output of these programs should appear in a new files called:
outputBruteForce.txt
naiveOutput.txt
output_enhanceddnc.txt
