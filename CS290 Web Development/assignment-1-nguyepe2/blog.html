My First Blog Post

Most of the challenges and troubles that I had with this assignment were related to how rusty I was with HTML. Just getting used to using tags after using other languages such as C++ and C took awhile for me. Something that also took me awhile to realize was that I needed to set permissions on the files so that I could view them on a browser.

Although I didn't really learn anything new from this assignment, I found that this assignment was a really good refresher about HTML.

The most useful resource that I used for this assignment was the HTML tutorials at w3schools.com. Because I used this website to help learn HTML when I was first introduced to it, I was able to use the website to quickly familiarize myself with HTML again.
